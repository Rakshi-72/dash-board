# Admin Dashboard (React + Tailwind)

A fully responsive admin dashboard with:
- Dashboard overview (KPIs + charts)
- User management (search, block/unblock)
- Product management (add/edit/delete + image preview)
- Orders management (status updates + details panel)
- Payments history with filters
- Analytics page with chart filters
- Settings page (profile + password)
- Global debounced search
- LocalStorage persistence (theme + last visited page)
- Toast notifications, loading skeletons, and smooth animations

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deploy the repository to **Netlify** or **Vercel**.
